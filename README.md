## Dripleaf

![](https://github.com/JNNGL/Dripleaf/actions/workflows/debug.yml/badge.svg)

### Building

1. Requirements:
   - CMake (>= 3.20)
   - C compiler
   - C++17 compatible compiler
   - Python 3

2. Compiling: <pre><code>$ git clone --recurse-submodules https://github.com/JNNGL/Dripleaf
$ cd Dripleaf
$ mkdir build && cd build
$ cmake ..
$ cmake --build .
</code></pre>
